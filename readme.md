12. <h3 id="12">一句话经济学</h3>

* 什么是利率？  
  利率就是货币的时间价值：    
  如果利率为负，也就是说钱越来越不值钱，就应该当下花掉或者换成其他保值的黄金白银之类的商品等。
  如果利率为正，那么就可以对比各个货币之间的利率，哪个利率高，就相当于把钱换成哪个币种好。

* 人民币汇率/汇价那么多种，傻傻分不清楚？  
  你就是人民币，那么以下一一对应：  
  在岸人民币： 你对自己的认可  
  人民币中间价：你朋友对你的认可  
  离岸人民币： 外面其他人对你的认可  
  人民币外汇牌价： 每个人心中对你的认可度  
  

* 美联储相关：

  你是公司管钱的财务。你们公司管理很先进，为了激发部门的工作热情，所有部门和你，部门和部门间都可以借钱和还钱，但是也会有对应的利息。你（央行）和老王老李老张  （公司各部门）之间借钱还钱的故事：
 
  **回购利率**：你从老王那里拿现金回来，付给老王的利息。把欠条给老王。  
  **隔夜逆回购利率（ON RRP）**：你只把钱借给老王一晚上的利息  
  **超额保证金利率/超额存款准备金利率**（IOER / Interest Rate on Excess Reserves）： 老王多赚的钱不知道怎么投资，都借给你，你给老王利息。 
  **存款准备金利率（IOR）**：安全起见，老王会把每月工资的固定比例借给你，你给老王的这部分钱的利息。  
  **商业银行存款利率**：老王把钱借给他手下员工的利息。  
  **美国联邦基金利率（EFFR）** = 美国同业拆借市场的利率 = 美国银行间隔夜拆借利率：  部门间也会互相借钱，隔壁老王和老李老张之间达成的夜里借钱第二天还钱的利息。        
  **贴现率**：老王借钱给了老李，但是老王现在缺钱了着急用钱老李又提前还不了，于是他把老李给他的没到期的欠条卖给老张，从老张这里拿到现金，老张愿意给他的利息。  
  **再贴现率**：老张拿着手里的老李的欠条，又来找你要换成现金，你愿意给老张的利息。  
  **美国国债收益率**：你要借钱，拿一张100块的借条给老王、老张、老李他们竞拍，最后老张以98中标拿走了你的100块借条，同时你得到了98块。2/98就是收益率，10年的借 条就是你长期的信用价值。也即美元价值。  

  邻居们都很聪明，都会做投资，不愿意借利息高的钱。所以，你会用隔夜逆回购利率来对比联邦基金利率，你希望市场上钱少的时候你就把隔夜逆回购利率设置低于联邦基金利率，大家就都直接问你借钱就可以了，不必借其他同行的钱，钱就会留在市场而不是回流美联储；当你把隔夜逆回购利率设置的高于联邦基金利率的时候，也就意味着大家都不愿意问你借钱，宁可问其他部门借钱也不问你借钱，借钱就相对要考量的东西多一些，市场上的钱就应该少。同时如果配合国债利率供给调节把市场上闲钱都吸纳到债券里，那么同业之前就将更加难借钱，从而稀释市场上的现金供给。
